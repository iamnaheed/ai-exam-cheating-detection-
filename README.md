
Title:
AI-Based Exam Cheating Detection System (Computer Vision + ML)

Overview:
This project detects suspicious exam behavior using facial landmark detection and a trained Random Forest classifier.

Features:

Real-time webcam monitoring

Head movement analysis

Multiple face detection

ML-based suspicious activity classification

Dataset:
Custom head movement dataset (<25MB)

Model:
Random Forest Classifier

How to Run:

pip install -r requirements.txt

python train_model.py

python main.py

Future Improvements:

Eye gaze tracking

Phone detection (YOLO)

Cloud dashboard

Deep learning upgrade
